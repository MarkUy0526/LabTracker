Behavior notes

  • Default on load: From = 1st of current month, To = today
  • X-axis labels: "Apr 1", "Apr 2" … formatted dates (works across month boundaries)
  • Each line: one equipment, colored with HSL spread
  • Filter button: destroys old chart instance, fetches new data, re-renders
  • Empty range: chart shows "No data for selected range" text
