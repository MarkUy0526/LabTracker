Behavior summary

  • Default on load: From = 1st of current month · To = today · Status = All
  • X-axis: One point per day in range (e.g. "Apr 1", "Apr 2"…)
  • Y-axis: Count of borrow requests on that day
  • Filter button: Destroys old chart, fetches new data, re-renders
  • Status = All: Counts all requests regardless of status
  • No data: Chart shows "No data for selected range" message
